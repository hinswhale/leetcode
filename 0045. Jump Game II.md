* 45. Jump Game II
* 难度：Medium｜ 中等
* 相关知识点：Array| Two Pointers | Greedy
* 题目链接：[https://leetcode.com/problems/jump-game-ii/description/](https://leetcode.com/problems/jump-game-ii/description/）

```python
class Solution:
    def jump(self, nums: List[int]) -> int:
        n = len(nums)
        far = 0 
        ans = 0
        next_i = 0
        for i in range(n-1):
            far = max(i + nums[i], far)
            if next_i == i:
                ans += 1
                next_i = far
        return ans
```
